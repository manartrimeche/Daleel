# Interface Daleel

Ce dossier contient les pages web servies par le backend FastAPI :

- `index.html` : interface chatbot
- `admin.html` : panneau d'administration
- `auth.html` : connexion et inscription
- `invite.html` : acceptation d'invitation
- `auth-guard.js` et `i18n.js` : scripts partagés

Le backend continue de les exposer via `/`, `/admin`, `/login`, `/register`, `/invite` et `/static/...`.
